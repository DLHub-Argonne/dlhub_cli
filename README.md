DLHub CLI
=========


A Command Line Interface for [DLHub](https://dlhub.org)

Source Code: https://github.com/DLHub-Argonne/dlhub_cli


## Installation

```
pip install dlhub_cli
```

## Development
After cloning this repository, run

```
pip install -e .
```
